## 📱 שימוש ראשון - 5 דקות

### 1️⃣ חבר מכשיר
```
- חבר Android למחשב
- לחץ "Detect Device"
- בדוק שהסטטוס: "Connected" ✅
```

### 2️⃣ חלץ קושחה
```
- File → Open Workspace
- בחר תיקייה
- לחץ "Extract Partitions"
- המתן... ☕
```

### 3️⃣ ערוך משהו
```
לשונית "Editor":
- העלה לוגו חדש
- או שנה build.prop
- צפה ב-"Preview"
```

### 4️⃣ יצא חבילה
```
- Tools → Generate Scatter File
- Tools → Export Modified Firmware
- קיבלת חבילה מוכנה לצריבה! ✅
```

### 5️⃣ צרוב (זהיר!)
```
⚠️ גבה קודם!
- פתח SP Flash Tool
- טען את ה-scatter
- צרוב
```

## 🆘 עזרה מהירה

### בעיה: התוכנה לא נפתחת
```bash
# נסה:
python main.py

# אם יש שגיאה, התקן תלויות:
pip install -r requirements.txt
```

### בעיה: מכשיר לא מזוהה
```
1. התקן MTK USB Drivers
2. נסה USB port אחר
3. הפעל כ-Administrator
4. בדוק Device Manager
```

### בעיה: שגיאה בצריבה
```
1. בדוק תאימות הקובץ למכשיר
2. בדוק MD5 checksums
3. נסה SP Flash Tool אחר
4. קרא FLASH_INSTRUCTIONS.txt
```

### עוד עזרה?
```
📖 קרא: USER_GUIDE.md → פתרון בעיות
📄 בדוק: logs/ → קבצי log למידע נוסף
```
