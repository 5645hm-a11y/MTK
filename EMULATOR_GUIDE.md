# Android Emulator Integration - הפעלת אמולטור אנדרואיד

## מה זה עושה?

התכונה החדשה מאפשרת להפעיל **אמולטור אנדרואיד אמיתי** עם המחיצות שחולצו מהמכשיר MTK.
במקום תצוגה גרפית מדומה, האמולטור מריץ את המערכת בפועל כמו מכשיר וירטואלי.

## דרישות מערכת

### אפשרות 1: Android Emulator (מומלץ)
1. התקן **Android Studio** מ-https://developer.android.com/studio
2. במהלך ההתקנה, וודא שמותקנים:
   - Android SDK Platform-Tools
   - Android Emulator
   - System Images (Android 11/12)

3. הגדר את משתני הסביבה:
   ```
   ANDROID_HOME = C:\Users\<שם_משתמש>\AppData\Local\Android\Sdk
   ANDROID_SDK_ROOT = C:\Users\<שם_משתמש>\AppData\Local\Android\Sdk
   ```

### אפשרות 2: QEMU (מתקדמים)
1. התקן QEMU מ-https://www.qemu.org/download/
2. הוסף את QEMU ל-PATH של המערכת

## איך להשתמש?

### שלב 1: חלץ או ייבא מחיצות
1. חבר מכשיר MTK ולחץ על **"חלץ מחיצות"**
   או
2. לחץ על **"ייבא מחיצות קיימות"** ובחר תיקייה עם קבצי .img

### שלב 2: הפעל את האמולטור
1. עבור ללשונית **"תצוגה מקדימה"**
2. לחץ על **"🚀 הפעל אמולטור אנדרואיד"**
3. האמולטור ייפתח בחלון נפרד ויטען את המחיצות שחולצו

### שלב 3: צפה במכשיר הווירטואלי
- האמולטור יאתחל את מערכת Android עם המחיצות שלך
- תוכל לראות את מסך האתחול, ממשק המשתמש, ואפליקציות
- המכשיר הווירטואלי פועל ממש כמו טלפון אמיתי

## מה האמולטור טוען?

האמולטור מקבל את המחיצות הבאות:
- **system.img** - מערכת ההפעלה Android
- **vendor.img** - דרייברים וספקים
- **boot.img** - Kernel ו-ramdisk
- **super.img** - Dynamic partitions (אם קיים)
- **userdata.img** - נתוני משתמש

## פתרון בעיות

### "Android SDK לא נמצא"
- וודא שהתקנת Android Studio
- בדוק את משתני הסביבה ANDROID_HOME
- נסה להפעיל מחדש את המחשב אחרי ההתקנה

### "האמולטור לא מאתחל"
- בדוק שמחיצת boot.img קיימת
- ודא שיש לפחות system או super partition
- בדוק את הלוגים לשגיאות

### "QEMU לא נמצא"
- התקן QEMU והוסף לPATH
- הפעל מחדש את הטרמינל/אפליקציה

## הערות חשובות

⚠️ **אזהרה**: האמולטור דורש משאבי מערכת:
- לפחות 4GB RAM פנויים
- מעבד עם תמיכה ב-virtualization (Intel VT-x / AMD-V)
- מקום דיסק עבור המחיצות

⏱️ **זמן אתחול**: האמולטור עשוי לקחת 2-5 דקות לאתחול מלא, במיוחד בפעם הראשונה.

🔧 **מצב מתקדם**: אם אתה מכיר QEMU, תוכל להריץ את המחיצות ישירות עם פרמטרים מותאמים אישית.

## יתרונות האמולטור

✅ **בדיקה אמיתית** - רואים בדיוק איך הקושחה תתנהג במכשיר
✅ **ללא סיכון** - בודקים שינויים לפני צריבה למכשיר פיזי
✅ **גישה מלאה** - יכול לבדוק אפליקציות, הגדרות, boot sequence
✅ **ניפוי באגים** - מזהה בעיות טרם התקנה במכשיר האמיתי
